# Arin 4K Direct-Use Godot Pack

## Copy this folder exactly to

`res://assets/characters/arin_4k/`

Then drag this ready scene into your level:

`arin_4k.tscn`

## Included animations

- idle
- run
- jump
- fall
- jump_fall
- kick
- charged_kick
- push_fail

## Input Map required

- move_left
- move_right
- jump
- kick
- charged_kick
- push_fail

## Files

- Four true 4096×4096 transparent master atlases
- Separate transparent 1024×1024 numbered frames
- Godot SpriteFrames resource
- Ready CharacterBody2D scene
- Ready movement/animation controller
- Animated preview GIFs

## Important

The files are direct-use and transparent. The original art was AI-generated, so
minor frame-to-frame costume or anatomy variation may still exist. The shared
alignment and animation order have been normalized for Godot.
